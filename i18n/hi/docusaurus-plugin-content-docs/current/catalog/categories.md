# जैम पर श्रेणियाँ
एक विक्रेता जीईएम पोर्टल पर उत्पादों या सेवाओं की पेशकश कर सकता है। जीईएम पोर्टल पर दोनों प्रकार की पेशकशों के लिए बहुत अधिक श्रेणियां उपलब्ध हैं।

**Product Categories**

```
Products

 ```

```

 ```
